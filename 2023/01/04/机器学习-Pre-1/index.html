二分分类

什么是二分分类

例如，给出一张图片。输出该图片中是否含有一只猫。如果含有，则输出1，如果没有，则输出0。即为一个二分分类。

以一个64*64像素的图片为例，其可以由RGB分别对应的三个64维的矩阵组成。为了表示这个图片，我们用一个64*64*3 = 12288维的特征向量x来记录下所有像素的RGB值。

则二分分类即要求我们根据这个12288维的输入x，预测输出标签y是1还是0

二分分类的符号表示

我们用(x,y)来表示一个样本，其中，x为该样本的的nx维特征，而y为0或1，代表对应的标签值。

用m来表示训练集中样本的个数，自然的，训练集可以被表示为这样的集合：

{(x(1), y(1)), (x(2), y(2)), ...... , (x(m), y(m))}

我们可以用一个nx*m维的矩阵X来表示所有样本点的features 同样的，用一个1*m维的矩阵Y来表示所有样本点的标签

Logistic Regression

对于一个二分分类问题而言，我们希望根据对应的x值输入来得到y的值，实际上，我们所计算出的是 = P(y = 1 | x)，也就是说y为1的概率，因此我们希望y的值是一个介于0～1之间的数。

Gradient Descent

我们可以采用梯度下降法（Gradient Descent）来寻找使最小的参数W和b。由于合理的定义了Loss function，是一个凸函数，存在一个全局的最优点（最小值）。因此我们可以随意为W和b定义一个初始值，而采用梯度下降法寻找到一个最佳点，通常可以定义为0或者一个随机值

Acitivation functions

回顾上述内容，当我们进行正向传播的过程时，都是采用sigmoid函数作为激活函数，即，实际上我们也可以采用其他的激活函数。

例如双曲正弦函数

该函数的值域为（-1，1），几乎就是sigmoid函数的平移，但是相较于sigmoid函数来说，tanh函数经过零点，且平均值为0，因此在大部分场合其性能要优于sigmoid函数（二分分类问题除外，因为二分分类问题的输出仅为0或1，故我们希望激活函数的输出在0和1之间）

同样的，不同层的激活函数可以不同，g[1]表示layer1使用了激活函数g，f[2]表示layer2使用了激活函数f

tanh函数和sigmoid函数有一个共同的缺点，即z非常大或非常小时，其斜率趋向于0，这会使梯度下降法的速度变得十分缓慢。

另一些激活函数，例如ReLU函数没有这样的问题，在机器学习的第一篇文章曾经提到过ReLu函数

ReLU函数是大多数情况下可以使用到的激活函数

有时并不希望z在负数时的值为0，这时也可以使用Leaky ReLU，其方程为a = max(0.01z,z)，这样在z<0时会是一个缓慢变化的曲线

Initial the weight randomly

归纳法可以证明，如果将W的值全部置为0，根据对称性，所有的Hidder neuron都在计算同一个函数，无论训练多长时间，他们都会进行同样的计算，因此超过1个的Hidden nueron都是没有意义的。为了打破这种对称性，应该随机初始化参数。

需要注意的是，为了防止初始化的参数落在类似sigmoid函数、tanh函数这样的激活函数较平缓的区域，一般我们将初始化的参数设定为一个很小的值，因此需要将random出来的值*0.01，作为我们初始化的参数

Deep Nueral Network

Notation

过去我们仅研究过单隐藏层，也就是2-layer的神经网络模型。对于含有多个隐藏层的神经网络模型，我们采取以下的符号约定

  • L = the number of layers

  • n[l] = the number of units in layer l

  • a[l] = the activations in layer l

    • a[0] = x
    • a[L] = y

Forward propogation for layer l

Input

  • a[l-1]

Output

  • a[l]
  • W[l],b[l]

Backward propagation for layer l

Input

  • da[l]

Output

  • da[l-1],dW[l],db[l]

Parameter and Hyperperemater

Parameters: W[1],b[1],W[2],b[2],...

Hyperparameters: Other parameters

  • learning rate
  • iterations
  • Hidden layers
  • Hidden units
  • choice of activation function
  • ...

Hyperparameters will control W & b
